## What is ds-start?

ds-start is the composable Next.js app kit. It scaffolds a production-ready foundation and lets you add only the modules you need — authentication, payments, email, file uploads, forms, CI, deploys, and more.

Agent workflows ship with the project so Claude, Codex, and other coding agents get conventions, quality gates, and handoff patterns from the start.

## Quick Start

```bash
npx ds-start init my-app
```

This scaffolds a new Next.js project with the full foundation stack. You'll be prompted to select optional modules.

To add a module to an existing project:

```bash
npx ds-start add email
```

## Foundation Stack

Every ds-start project includes:

- **Next.js 16** — App Router, React Server Components, Turbopack
- **TypeScript** — Strict mode, zero `any` tolerance
- **Tailwind CSS 4** — Utility-first styling
- **shadcn/ui** — Accessible component primitives
- **Bun** — Fast package management and scripts
- **oxlint + oxfmt** — Fast, consistent code quality
